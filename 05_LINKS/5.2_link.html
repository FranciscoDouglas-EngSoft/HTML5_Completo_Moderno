Exemplos da Tag LINK

A tag <link> é usada no <head> para conectar recursos externos:

  • Stylesheet: CSS externo
  • Icon: Favicon e ícones
  • Preload: Pré-carregar recursos
  • Prefetch: Pré-buscar próxima página
  • DNS-prefetch: Resolver DNS antecipadamente
  • Canonical: URL canônica para SEO

Esta página usa várias dessas funcionalidades!
